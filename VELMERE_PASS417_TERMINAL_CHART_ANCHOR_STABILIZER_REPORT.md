# PASS417 — Terminal Chart Anchor Stabilizer

PASS417 is a stability-first implementation pass for Velmère Shield, Velmère Browser/Lens and Velmère Real Markets.

## Implemented

- Browser/Lens search remains input-anchored with a hard maximum of 3 visible suggestions.
- Shield search uses the same PASS417 clamp and runtime close event.
- Real Markets expands the provider-ready universe with indices, FX, commodities, REIT and ETF lanes.
- Real Markets modal remains chart-first and object-safe. Basic / Pro / Advanced render deterministic 10 / 14 / 20 fields.
- Orbit 360 remains parked behind a crash boundary until it can be reintroduced as a lazy isolated component.
- Chart drag is treated as a stability contract: dead-zone first, reset baseline per gesture, no first-frame jump.
- PDF parity keeps HTML preview and downloaded PDF on one resolved payload, one locale and one field order.
- Security and Research Lab copy stay simple, public and boundary-based.

## Validation intent

PASS417 does not claim fully live market coverage. It prepares provider adapters, source age, second-source drift, checksum, official icon/logo lanes and reconnect-aware chart semantics.

## Result

Terminal Chart Anchor Stabilizer keeps the product stable while Real Markets moves closer to Shield parity. PDF parity and search anchoring remain the primary release blockers being closed.
