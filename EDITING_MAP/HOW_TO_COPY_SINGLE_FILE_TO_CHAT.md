# Jak wysyłać pojedyncze pliki do edycji

1. Otwórz plik w VS Code.
2. Skopiuj całość `Ctrl+A` → `Ctrl+C`.
3. Wyślij mi wiadomość: “Edytujemy tylko [sekcja]. Oto plik: ...”.
4. Wklej zawartość pliku.

Najczęstsze przypadki:

- Square: `components/square/VelmereSquareClient.tsx`
- Login: `components/auth/AuthGate.tsx`
- Account: `components/dashboard/DashboardClient.tsx`
- VLM popup: `components/vlm/VlmModeChoicePrompt.tsx`
- VLM Pro: `components/vlm/VlmProTerminal.tsx`
- Header: `components/Navbar.tsx`
- Koszyk: `components/CartDrawer.tsx`
- Główna strona: `components/home/HomePageClient.tsx`

Jeżeli chcesz zmieniać teksty językowe, dorzuć też:

- `messages/en.json`
- `messages/pl.json`
- `messages/de.json`
