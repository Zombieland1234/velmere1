# PASS207 — AI Brain Decision Dock

## What changed

- Added a compact **Decision Dock** to the VLM Brain tile drawer.
- The clicked tile now shows four operator-ready cards: **Priority**, **Confidence limit**, **Source mode** and **Review window**.
- Missing, fallback or partial data now visibly caps confidence inside the drawer instead of looking like a clean pass.
- The dock is localized for PL/EN/DE and stays inside the existing portal drawer, so it does not create another modal or Source Spine.
- Public QA/FPS/WebGL clutter remains hidden behind PASS206 gates.
